# Security Policy

Do not report credentials, private IP addresses, public IP addresses, private
media names, or logs containing secrets in public issues.

This project currently does not accept private vulnerability reports by email.

If you find a security-sensitive issue, open a minimal public issue without
secrets, or wait until a private reporting channel is published.

## SMB1 security notice

SMB1 is a legacy protocol. This project is intended for trusted local networks
and legacy local media storage only.

Do not expose SMB1 shares to the public internet.
